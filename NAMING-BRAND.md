## 一、品牌定位三角

### 功能核心（一句话）

输入汉字 → 看拼音、观笔顺动画、手写跟练。一个安静专注的汉字书写练习工具，不做词典、不做课程、不做社区。

### 情感价值（一句话）

**一笔一画，重新安放你与汉字之间「会读不会写」的距离。**

### 品牌人格（三个形容词）

| 人格 | 含义 |
|------|------|
| **克制** | 只做「写字」这一件事，不多加一个功能。不弹广告、不推课程、不啰嗦。 |
| **专业** | 笔顺数据来源权威、拼音方案准确、多音字有据可查。不是游戏，是工具。 |
| **文雅** | 宣纸留白、墨黑朱红、米字格素净克制。不是卡通风，不是科技冷感。 |

### 竞品差异化

| 竞品 | 定位 | 差异化机会（我们的品牌打什么） |
|------|------|-------------------------------|
| **都会识字** | 儿童识字 App，卡通界面 + 课程体系 | 我们覆盖成人+儿童，不做卡通/课程，用东方极简与"书法素养"拉开差距。 |
| **快记漢語** | 闪卡记忆/速成汉语，强调"快" | 我们反其道——不追求快，强调「一笔一画安静写」。快记解决"记不住"，我们解决"不会写"。 |
| **Skritter** | 国际化写汉字 App，SaaS 订阅，词库驱动 | 我们不是订阅制（一次买断），不依赖词库体系，回归单字练习的纯粹体验。 |
| **Pleco** | 超级词典，什么都查 | 我们不做词典，不做 OCR，不做例句——只做铅笔+米字格这一件事。Pleco 是字典，我们是习字本。 |
| **洪恩识字** | 儿童游戏化学字，动画+故事 | 完全不一个赛道。我们不游戏化、不故事化，用"书写本身的美感"吸引用户。 |

### 候选 1：一格

| 字段 | 内容 |
|------|------|
| **名称本体** | **一格** |
| **英文名推荐** | OneGrid / YiGe |
| **词源/语义脱胎** | 从已锁定的「米字格」Logo 推出。格 = 书写的基本空间单位。一个汉字占一格，每一笔都在格中。「一格」既有字面意义（一个米字格），也有隐喻意义（"做事有格调"）。 |
| **隐喻层次** | 意识到：汉字写在一个格子里。联想到：格物致知的"格"——认真对待每一个字。文化原型：田字格、米字格是每个中国人学写字时都见过的视觉母语。 |
| **品牌人格区隔** | vs 都会识字：不做"游戏闯关"，就一个格子，写就完了。vs Skritter：不搞订阅词库，每字一格，简洁到底。 |
| **iOS App Icon 潜力** | ★★★★★ 极强。"一格"二字可作为 wordmark，也可用单格 + 朱砂红一点作图形标，与 logo（永+米字格）天然呼应。字体设计空间大。 |
| **与 Logo 契合度** | **10 / 10**。米字格就是"格"，logo 中「永在格中」与名字「一格」一一对应。不可能更契合。 |
| **潜在问题** | ① 搜索"一格"在 App Store 可能被其他 App 占位。② 出海时 "OneGrid" 需解释语境。③ 部分用户可能联想"一格生物/一格英语"等相关品牌。 |
| **App Store 上架可行性** | ⚠️ 中等风险。"一格"作为独立词在 ASO 上竞争较激烈，但品牌独特性足够。建议配副标题（如"一格·汉字书写"）做 ASO 兜底。 |

## 三、推荐排序

### 🥇 #1 推荐：一格

**为什么是它？**

从已锁定 logo 出发，没有比"一格"更直接的名字。Logo = 永字 + 米字格，名字 = 一格。名字解释 logo（一个格子），logo 解释名字（永在格中），双向咬合，形成品牌记忆锚点。从用户场景出发，输入一个汉字 → 一个格子 → 一笔一画写完。整个体验被压缩进两个字。"一格"也天然适合 wordmark 设计，"一"字可以拉长成米字格的横线，"格"字保留木字旁的结构感。品牌空间大：可以延伸"一格书"（写出来的作品）、"一格册"（收藏集）。出海名 OneGrid 在英文语境也干净可理解。唯一风险是 ASO 竞争，但品牌独特性足以对冲。

---

### 🥈 #2 推荐：点墨

**为什么是它？**

如果"一格"过于直白，"点墨"提供了品牌温度。从 logo 锚点出发：朱砂红的一笔/一点 = 点墨。从场景出发：每次落笔都是"点下一墨"。从情感出发：从"胸无点墨"到"心中有墨"——产品的情感价值恰好是填补"会读不会写"的 gap。点墨的名字自带语言延伸能力："点墨一刻"（专注书写的时间）、"点墨集"（收藏合集）。作为 iOS icon，"点墨"二字或一个红点图形，在 App Store 多图标的拥挤海洋里，最有可能被一眼认出。品牌温度高于"一格"，更适合走 freemium 向 premium 的转化叙事。

---

### 🥉 #3 推荐：八法

**为什么是它？**

如果你的目标是做一个有文化深度的品牌，而不是一个工具，"八法"是最好的选择。从 logo 锚点出发：永字八法是 logo 的完整解释——名字和 logo 构成闭环。从品类出发：没有一个汉字学习工具叫"八法"——它在所有竞品名字中独一无二。从品牌溢价出发：当你叫"八法"，你的定价可以比"写字 App"更高——因为你在卖方法论，不是卖工具。风险也很清楚：新用户第一反应可能是"八法是什么？"——需要一定的市场教育和 ASO 副标题辅助（如"八法·永字入门"）。但如果品牌希望走向文化消费（文创周边、书法工具、线下工坊），"八法"是最好的长期品牌资产。

## 四、必须避免的名称词汇（黑名单）

| 禁用词 | 为什么避免 |
|--------|-----------|
| **学**（如 x学、学x） | 被教育产品用烂：有道、学而思、流利说..."学"字让人联想到课程序列、作业、考试压力。我们的品牌人格是"安静的写字工具"，不是"学习系统"。 |
| **课**（如 x课、课x） | 产品不做课程体系。"课"暗示了你需要"上课"才能用——这与"打开即写"的轻量体验冲突。 |
| **帮**（如 xx帮） | 过度工具化表达，暗示社区/互助。"帮"字在 App Store 已被拼音学习类产品大量占据，品牌区隔困难。 |
| **快**（如 快x、x快） | 品牌人格主打"安静/专注/一笔一画"，与"快"完全冲突。快记漢語已经在快这个维度上占了，不必正面竞争。 |
| **神**（如 神x、x神） | 儿童产品用语（神笔马良）。产品气质是克制/文雅，不是"神奇/炫酷"。 |
| **通**（如 一通、x通） | 被各类学习产品用滥（学通、一通、融通），语义空洞，无法传递"书写"的质感。 |
| **师**（如 汉字师、x师） | 暗示教学角色（名师、导师）。产品定位是工具，不是老师——"师"字给用户"又要上课"的心理负担。 |
| **典**（如 字典、x典） | 脱胎于"字典"。"这不是词典"在 PRD 中已明确。用"典"让用户混淆：你到底是查字的还是写字的？ |
| **速**（如 速写、x速） | "速写"在艺术领域有特定含义（快速写生）。且"速"与品牌反效率的安静气质矛盾。 |
| **萌/趣/乐** | 卡通/儿童向用词。logo 方向已锁定东方极简+朱砂红，"萌趣"色彩一秒毁掉品牌调性。 |

## 五、补充说明

### 与上一轮废名的关系

- **声笔 ToneStroke**：排除原因确认为"看不懂、太 generic"。本报告推荐的名字（一格、点墨、八法）全部避免了"二字拼接一个含义模糊的合成词"的问题——每个名字都是一个**完整意象**，不需要"声调+笔顺"这种功能拆解来理解。
- **永字/起笔/一笔/临帖/字源/写字**：这些词被上一轮面试拒绝的原因不是词本身不好，而是"凭感觉拍"。本报告每个名字都从品牌三角和两个锚点出发做了完整推理。例如"一格"虽包含"一"字（与"一笔"同源），但推理由米字格支撑，结构完整，不是凭感觉选词。

### 与 logo 的视觉协同建议

不论选择哪个名字，建议：
1. **App Store icon** = 优先用图形（永+米字格+朱砂红一笔）+ 名字做副标
2. **启动屏/品牌页** = 名字 + 米字格的文字组合 logo
3. 所有名字都适合与 logo 中的"永"字共存（例如：一格·写在永里 / 点墨·永字入门）
